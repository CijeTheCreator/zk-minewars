ZK Minewars is a fully on-chain, two-player competitive Minesweeper game built on the Stellar/Soroban smart contract platform, powered by zero-knowledge proofs to ensure fair, trustless gameplay. Two players go head-to-head on a shared 9×9 board where each player secretly commits their own mine layout — meaning neither can see the other's placement, but the contract can cryptographically verify that every tile reveal is honest. Players can optionally stake MineBucks on the outcome, held in escrow and paid out automatically to the winner.

Players alternate turns, each selecting a tile to reveal on the opponent's board and submitting a ZK proof confirming whether it's a mine or a safe tile. These proofs are verified entirely on-chain, making it impossible to cheat. Each player has a configurable number of lives — hitting a mine costs the opponent a life and scores the attacker a point. The game ends when lives run out or the round limit is reached, with the winner determined by remaining lives and score. Time windows keep the game moving, and if a player goes idle past their window, their opponent can claim the win by forfeit.
